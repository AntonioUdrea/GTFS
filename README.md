# Bistrița-Năsăud County GTFS feeds
Here you can find the GTFS feeds for all the companies that operate routes inside the county, these being:
- SC Transmixt SA: for the routes operated in the Bistrița Municipality
- SC Local Serv Năsăud SRL: for the routes operated in the Năsăud city.
- SC Transport Urban Beclean SRL: for the routes operated in the Beclean city.
- Primăria Sângeorz-Băi: for the routes operated in the Sângeorz-Băi city.
- Consiliul Județean Bistrița-Năsăud: for the routes operated throughout the county.

Each feed contains two options, viewing it in a readable HTML format (thanks to this GTFS to HTML convertor: https://run.gtfstohtml.com/) or downloading the .zip archive.
